## 🚀 Быстрый старт (Развертывание)

**Локальный запуск (Windows/macOS/Linux):**
```bash
git clone https://github.com/your-username/laser-workshop-crm.git
cd laser-workshop-crm
python -m venv venv
source venv/bin/activate
pip install -r backend/requirements.txt
cp .env.example backend/.env
python backend/main.py
```
*Интерфейс:* `http://localhost:8000` | *API Swagger:* `http://localhost:8000/docs`

**Установка на сервер (Linux / Raspberry Pi):**
```bash
cd laser-workshop-crm
chmod +x deploy/*.sh
sudo ./deploy/install.sh
```

## 🎯 Почему Laser CRM? (Решение проблемы)
Вместо использования 4-х разных программ (Excel для расчетов, VK для общения, Trello для заказов, МойСклад для учета), мастерская получает единый интерфейс. Расчет цены (по вектору/площади), общение с клиентом и автоматическое списание материалов (в см²) происходят **в одном окне**.

## ✨ Ключевые возможности

*   🧮 **Умный калькулятор (11 алгоритмов):** Серверный расчет стоимости по площади, длине реза, времени работы станка, объему (3D клише), весу и комбинированным методам. Встроена защита от демпинга (минимальная цена заказа) и автоматический оптовый дисконт.
*   🤖 **Интеграция ВКонтакте (VKBottle) & Live-Чат:** Бот перехватывает сообщения, автоматически создает карточку клиента в базе и позволяет менеджеру вести переписку прямо из интерфейса CRM без перезагрузки страницы.
*   🌴 **Режим "Отпуск":** Системный тумблер, активирующий умный автоответчик для новых клиентов на период отсутствия мастера.
*   📈 **AI-прогноз выручки:** Модуль машинного обучения (`scikit-learn` / Linear Regression) анализирует финансовые тренды за последние 30 дней и математически прогнозирует доход на завтра.
*   💎 **Умная программа лояльности:** Динамическое начисление кэшбэка (5%) при переводе заказа в статус «Выдан» с автоматической записью баланса в профиль клиента.
*   🏭 **Складской учет (ERP):** Контроль остатков материалов (листовая фанера, акрил, фурнитура) с системой алертов для позиций, требующих срочной закупки (`low-stock`).
*   💾 **Резервное копирование:** Мгновенная генерация и скачивание ZIP-архивов, включающих дампы базы данных SQLite и CSV-экспорт таблиц (клиенты, заказы).
*   🔒 **Enterprise Безопасность:** JWT-авторизация (OAuth2), хэширование паролей (`bcrypt`), строгая валидация входящих данных через Pydantic v2 и защита от XSS/SQLi.

## 💻 Локальный запуск (Среда разработки на ПК)

Для развертывания проекта на локальном компьютере (Windows/macOS/Linux) для написания и тестирования кода:

**1. Клонирование и настройка окружения:**
```bash
git clone https://github.com/your-username/laser-workshop-crm.git
cd laser-workshop-crm

# Создание и активация виртуального окружения
python -m venv venv
source venv/bin/activate  # Для Windows используйте: venv\Scripts\activate

# Установка зависимостей
pip install -r backend/requirements.txt
```

**2. Настройка переменных окружения:**
Создайте файл `.env` на основе шаблона:
```bash
cp .env.example backend/.env
```
Отредактируйте `backend/.env`, указав ваш токен ВКонтакте и придумав сложный `SECRET_KEY`.

**3. Запуск сервера:**
```bash
python backend/main.py
```
*(При первом запуске база данных и пользователь-администратор будут созданы автоматически).*

**Точки входа:**
*   **Фронтенд (Интерфейс CRM):** `http://localhost:8000`
*   **Документация API (Swagger UI):** `http://localhost:8000/docs`
*   **Доступ по умолчанию:** Логин `admin`, Пароль `admin` *(Обязательно смените пароль в настройках!)*

## 🍓 Развертывание в Production (Raspberry Pi / Linux VPS)

Благодаря директории `deploy/`, установка на чистый Linux-сервер (Debian/Ubuntu/Raspbian) занимает **около 2-х минут**.

**Шаг 1. Перенос проекта**
Склонируйте репозиторий на сервер:
```bash
cd /home/pi
git clone https://github.com/your-username/laser-workshop-crm.git
cd laser-workshop-crm
```

**Шаг 2. Запуск инсталлятора**
Сделайте скрипты исполняемыми и запустите процесс установки от имени Root:
```bash
chmod +x deploy/*.sh
sudo ./deploy/install.sh
```

**Что делает скрипт установки:**
1. Устанавливает системные зависимости (`python3-venv`, `nginx`, `ufw`).
2. Создает изолированное виртуальное окружение и ставит Python-библиотеки.
3. Настраивает **Nginx** как обратный прокси на порт 80.
4. Создает демона в **Systemd** (`lasercrm.service`) для автозапуска бэкенда при загрузке системы.
5. Настраивает брандмауэр **UFW** (открывает только порты 22 SSH и 80 HTTP).

**Шаг 3. Управление сервером:**
Для удобного администрирования используйте готовые утилиты:
*   `./deploy/status.sh` — Мониторинг статуса приложения, загрузки процессора и использования ОЗУ.
*   `./deploy/start.sh` — Корректный перезапуск системы после обновления кода (`git pull`).
*   `./deploy/stop.sh` — Полная остановка приложения.

🎉 **Готово!** Теперь CRM доступна по локальному или белому IP-адресу вашего сервера.

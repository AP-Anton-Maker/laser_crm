# 🔬 Laser CRM: Интеллектуальная система управления лазерной мастерской

[![Python](https://img.shields.io/badge/Python-3.9+-blue.svg)](https://www.python.org/)
[![FastAPI](https://img.shields.io/badge/FastAPI-0.109+-green.svg)](https://fastapi.tiangolo.com/)
[![SQLAlchemy](https://img.shields.io/badge/SQLAlchemy-2.0-red.svg)](https://www.sqlalchemy.org/)
[![License](https://img.shields.io/badge/License-MIT-yellow.svg)](LICENSE)

**Laser CRM** — это современная, высокопроизводительная CRM-система для автоматизации процессов лазерной резки и гравировки. Проект объединяет в себе мощный асинхронный бэкенд на **FastAPI**, модульный фронтенд на **Vanilla JS**, интеграцию с **ВКонтакте** и элементы **Искусственного Интеллекта** для прогнозирования выручки.

Система разработана с учетом требований безопасности (JWT авторизация, валидация цен на сервере) и предоставляет полный цикл управления: от расчета стоимости заказа до складского учета и аналитики.

---

## ✨ Ключевой функционал

### 🧮 Умный калькулятор (Smart Calculator)
Серверная реализация 11 алгоритмов расчета стоимости (площадь, периметр, гравировка по времени, 3D-клише и др.).
- **Безопасность:** Полный пересчет цены на бэкенде исключает подмену стоимости со стороны клиента.
- **Гибкость:** Автоматическое применение оптовых скидок (до 20%) в зависимости от тиража.

### 🤖 VK-бот и Режим отпуска
Полноценная интеграция с социальными сетями через библиотеку `vkbottle`.
- **Автоответы:** Бот мгновенно реагирует на сообщения клиентов, сохраняя историю переписки в БД.
- **Режим отпуска (Vacation Mode):** Уникальная фича! Владелец может включить режим отпуска в настройках CRM. Бот автоматически начнет отвечать клиентам кастомным сообщением с датой возвращения, пока вы отдыхаете.

### 📈 AI-Прогнозирование и Аналитика
Встроенные механизмы бизнес-аналитики на базе `scikit-learn`.
- **ML-Прогнозы:** Линейная регрессия анализирует выручку за последние 30 дней и предсказывает доход на завтра.
- **Дашборды:** Визуализация ключевых метрик (LTV, средний чек, распределение по услугам) в реальном времени.

### 💎 Система лояльности (LTV & Cashback)
Автоматизированная программа удержания клиентов.
- **Кэшбэк:** Автоматическое начисление 5% от суммы заказа при завершении работы.
- **Сегментация:** Динамическое изменение статуса клиента (`new` → `regular` → `loyal` → `vip`) в зависимости от общей суммы покупок (LTV).

### 🏭 Склад и Бэкапы
- **Контроль остатков:** Система предупреждает о товарах, достигших минимального порога (`low-stock`).
- **Экспорт данных:** Выгрузка заказов и клиентской базы в CSV (совместим с Excel).
- **Резервное копирование:** Мгновенное создание бэкапов SQLite базы данных с возможностью скачивания одним кликом.

## 🛠 Стек технологий

| Компонент | Технологии |
| :--- | :--- |
| **Backend** | Python 3.9+, FastAPI, SQLAlchemy 2.0 (Async), Pydantic v2 |
| **Database** | SQLite (aiosqlite) |
| **AI & Math** | Scikit-learn, NumPy |
| **Integrations** | VKBottle (VKontakte API), PyJWT, Passlib (bcrypt) |
| **Frontend** | Vanilla JS (ES6 Modules), CSS3 (Variables, Grid/Flex), Chart.js |
| **Architecture** | REST API, Modular Monolith, Async/Await |

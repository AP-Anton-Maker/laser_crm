# 🔬 Laser CRM: Enterprise-система для лазерной мастерской

[![Python](https://img.shields.io/badge/Python-3.9+-blue.svg)](https://www.python.org/)

[![Platform](https://img.shields.io/badge/Platform-Raspberry%20Pi-green.svg)](https://www.raspberrypi.org/)

**Laser CRM** — это высокопроизводительная, асинхронная CRM-система, разработанная специально для автоматизации бизнеса в сфере лазерной резки, гравировки и ЧПУ-фрезеровки. 

Проект спроектирован по принципам **Clean Architecture** (Чистая Архитектура) как Single Page Application (SPA). Бэкенд и фронтенд полностью разделены. Архитектура оптимизирована для работы на микрокомпьютерах (Raspberry Pi 3B+/4, Orange Pi) без использования тяжелых Docker-контейнеров, обеспечивая максимальное быстродействие при минимальном потреблении ОЗУ.

## ✨ Ключевые возможности

*   🧮 **Умный калькулятор (11 алгоритмов):** Серверный расчет стоимости по площади, длине реза, времени станка, объему (3D), весу и комбинированным методам. Встроена защита от демпинга (минимальная цена) и автоматический оптовый дисконт.
*   🤖 **VK-бот (VKBottle) & Чат:** Полноценная интеграция с ВКонтакте. Бот перехватывает сообщения, автоматически создает карточку клиента в базе и позволяет менеджеру вести переписку прямо из интерфейса CRM.
*   🌴 **Режим "Отпуск":** Системный тумблер, активирующий умный автоответчик для новых клиентов на период отсутствия мастера.
*   📈 **AI-прогноз выручки:** Модуль машинного обучения (`scikit-learn` / Linear Regression) анализирует финансовые тренды за последние 30 дней и прогнозирует доход на следующий день.
*   💎 **Умная программа лояльности:** Динамическое начисление кэшбэка (5%) при переводе заказа в статус «Выдан» с автоматической записью в профиль клиента.
*   🏭 **Складской учет:** Контроль остатков материалов (фанера, акрил, краска) с системой алертов для позиций, требующих срочной закупки.
*   💾 **Резервное копирование:** Генерация и скачивание "на лету" ZIP-архивов, включающих дампы базы данных SQLite и CSV-экспорт таблиц.
*   🔒 **Enterprise Безопасность:** JWT-авторизация (OAuth2), хэширование паролей (`bcrypt`), строгая валидация входящих данных через Pydantic v2.

## 📂 Архитектура и структура проекта

Проект разделен на независимые модули. Фронтенд общается с бэкендом исключительно через REST API.

```text

│

│   ├── install.sh              # Авто-установка (Nginx, Systemd, UFW)
│   ├── start.sh                # Запуск сервисов
```

## 🛠 Технологический стек

**Backend:**
*   **Язык:** Python 3.9+
*   **Фреймворк:** FastAPI (Starlette + Pydantic)
*   **База данных:** SQLite + `aiosqlite`
*   **ORM:** SQLAlchemy 2.0 (Асинхронный режим)
*   **AI / ML:** Scikit-learn, NumPy
*   **Интеграции:** VKBottle (VK API)

**Frontend:**
*   **Архитектура:** SPA (Single Page Application)
*   **Стек:** HTML5, CSS3, Vanilla JavaScript (ES6+)
*   **Иконки:** FontAwesome

**DevOps & Инфраструктура:**
*   **ОС:** Linux (Raspberry Pi OS / Ubuntu / Debian)
*   **Веб-сервер:** Uvicorn (ASGI) + Nginx (Reverse Proxy)
*   **Управление процессами:** Systemd
*   **Безопасность:** UFW (Uncomplicated Firewall)

## 💻 Локальный запуск (Среда разработки)

Для развертывания проекта на локальном компьютере (Windows/macOS/Linux) для написания кода:

1. **Клонирование и настройка окружения:**
   ```bash
   git clone https://github.com/ваше-имя/laser-workshop-crm.git
   cd laser-workshop-crm
   
   python -m venv venv
   source venv/bin/activate  # Для Windows: venv\Scripts\activate
   pip install -r backend/requirements.txt
   ```

2. **Настройка переменных окружения:**
   Скопируйте шаблон конфига:
   ```bash
   cp .env.example backend/.env
   ```
   Отредактируйте `backend/.env`, указав ваш токен ВКонтакте и придумав сложный `SECRET_KEY`.

3. **Запуск сервера:**
   ```bash
   python backend/main.py
   ```
   *При первом запуске база данных и пользователь-администратор будут созданы автоматически.*
   * Фронтенд: `http://localhost:8000`
   * Документация API (Swagger): `http://localhost:8000/docs`
   * Доступ по умолчанию: Логин `admin`, Пароль `admin` (Обязательно смените пароль в продакшене!).

## 🍓 Развертывание в Production (Raspberry Pi / VPS)

Благодаря директории `deploy/`, установка на чистый Linux-сервер занимает **около 2-х минут**.

1. Перенесите проект на сервер.
2. Сделайте bash-скрипты исполняемыми:
   ```bash
   chmod +x deploy/*.sh
   ```
3. Запустите инсталлятор от имени Root:
   ```bash
   sudo ./deploy/install.sh
   ```
   **Инсталлятор автоматически:**
   * Установит `python3-venv`, `nginx`, `ufw`.
   * Создаст виртуальное окружение и установит библиотеки.
   * Настроит Nginx как обратный прокси на порт 80.
   * Создаст демона в Systemd (`lasercrm.service`) для запуска бэкенда при загрузке системы.
   * Настроит файрвол (разрешит только порты 22 и 80).

4. **Управление сервером:**
   * `./deploy/status.sh` — Посмотреть статус приложения, загрузку CPU и RAM.
   * `./deploy/start.sh` — Перезапустить систему после обновления кода.
   * `./deploy/stop.sh` — Остановить приложение.

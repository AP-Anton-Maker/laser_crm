📑 ПОЛНЫЙ ОТЧЕТ О РАЗРАБОТКЕ CRM «ЛАЗЕРНАЯ МАСТЕРСКАЯ»
🎯 Цель проекта
Создание полнофункциональной CRM-системы для лазерной мастерской с управлением заказами, клиентами, складом, интеграцией с ВКонтакте и AI-аналитикой.
Стек: Python (FastAPI), SQLAlchemy 2.0 (Async), SQLite, Vanilla JS (ES6 Modules), VKBottle, Scikit-learn.

_✅ Реализованный функционал (по шагам)_
_1. Фундамент и База Данных

    Асинхронность: Полностью асинхронный стек (asyncio, aiosqlite, SQLAlchemy 2.0).
    Модели данных:
        User: Сотрудники с ролями.
        Order: Заказы с параметрами (JSON), статусами, датами.
        Client: Клиенты с сегментацией (New, Regular, VIP), LTV и балансом кэшбэка.
        Inventory: Склад с порогами минимального остатка.
        PromoCode: Система промокодов.
        ChatMessage: История переписки с ВК.
        CashbackHistory: Журнал операций лояльности.
        AuditLog: Журнал действий пользователей.

_2. Бизнес-логика и Безопасность_

    Smart Calculator (services/calculator.py):
        Реализовано 11 алгоритмов расчета (площадь, периметр, гравировка по времени, 3D-клише и т.д.).
        Защита от подмены: Сервер пересчитывает цену заказа. Если цена с фронтенда отличается от расчетной (>2 руб), заказ отклоняется (HTTP 400).
        Оптовые скидки: Автоматическое применение скидок до 20% за тираж.
        Безопасность: Защита от деления на ноль, валидация отрицательных чисел.

_3. Система Лояльности (LTV & Cashback)_

    Автоматическое начисление: При завершении заказа (status=DONE) клиенту начисляется 5% кэшбэка.
    Сегментация: Автоматическое повышение статуса клиента (new → regular → loyal → vip) в зависимости от суммы покупок (LTV).
    История: Полная аудиторская запись всех начислений и списаний.

_4. Интеграция ВКонтакте (VKBottle)_

    Фоновый бот: Запускается внутри процесса FastAPI через lifespan.
    Входящие: Сообщения от клиентов сохраняются в БД, клиент получает автоответ.
    Исходящие: Менеджер может отвечать клиенту из интерфейса CRM → сообщение уходит в ВК и сохраняется в историю как "от админа".

_5. Аналитика и ИИ_

    Дашборд: Общая выручка, средний чек, долги по кэшбэку, топ-5 клиентов.
    Графики: Выручка по услугам, распределение по сегментам.
    Календарь: Заказы, сгруппированные по датам выполнения.
    AI Прогноз: Использование scikit-learn (Linear Regression) для предсказания выручки на завтра на основе трендов последних 30 дней. Определение тренда (Рост/Падение/Стабильно).

_6. Складской учет_

    Low Stock Alert: Эндпоинт /low-stock возвращает товары, требующие закупки (quantity <= min_quantity).
    Учет материалов: Полный CRUD позиций, поставщиков и цен.

_8. Фронтенд (Vanilla JS)_

    Модульность: Код разбит на ES6 модули (import/export).
    Роутинг: Переключение вкладок без перезагрузки страницы.
    UI Компоненты: Динамическая отрисовка таблиц заказов, уведомления (Toasts).
    Связь с API: Асинхронные запросы (fetch) с обработкой ошибок.

_🚀 Как запустить проект_

    Установка зависимостей:
```
cd backend
pip install -r requirements.txt
```
Настройка переменных окружения:
Создайте файл .env в папке backend (или экспортируйте переменные):
```
VK_TOKEN=ваш_токен_группы_вк
DATABASE_URL=sqlite+aiosqlite:///../data/laser_crm.sqlite3
```
(Если токен не указан, бот просто не запустится, остальная система будет работать).
Запуск сервера:
```
cd backend
python main.py
```
    Сервер запустится на http://0.0.0.0:8000.
    База данных и папка data создадутся автоматически при первом старте.
    Запуск фронтенда:
    Просто откройте файл frontend/index.html в браузере (или через Live Server в VS Code).
    Убедитесь, что в JS файлах указан правильный адрес API (http://localhost:8000/api).

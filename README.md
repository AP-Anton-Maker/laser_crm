# Laser CRM — Умный помощник для лазерной мастерской

**Laser CRM** — это готовая система для автоматизации небольшой лазерной мастерской (резка, гравировка, маркировка). Проект объединяет:

- 🤖 **VK-бота** (умный помощник) с 11 типами калькулятора, FSM-диалогами, кэшбэком и промокодами
- 🖥️ **Админку Django** с удобным управлением заказами, клиентами, складом и чатом
- 🐳 **Docker** для простого развёртывания на Raspberry Pi или любом сервере

Система не требует внешних интеграций (ЮKassa, СДЭК и т.п.) — всё общение с клиентом идёт через VK, а оплата/доставка решаются в переписке. Это идеальное решение для **одного мастера**, который хочет автоматизировать приём заказов и расчёт стоимости.

---

## ✨ Возможности

### Для клиента (в VK)

- **Каталог услуг** – список доступных работ с ценами
- **Рассчёт стоимости** – 11 типов калькулятора (площадь, длина реза, глубина гравировки, цилиндр, 3D и др.)
- **Баланс кэшбэка** – начисление 5% за каждый выполненный заказ
- **Связаться с мастером** – отправить вопрос или макет
- **Умный помощник** – дружелюбный диалог с подсказками и эмодзи

### Для мастера (админка Django)

- **Управление заказами** – канбан-доска (новые, в работе, готовы, выданы)
- **Клиенты** – автоматическое создание из VK, история заказов и кэшбэка
- **Склад материалов** – учёт остатков с контролем минимального количества
- **Промокоды** – создание и управление скидками
- **Чат с клиентом** – удобный интерфейс как в мессенджере (отправка/получение сообщений)
- **Настройки** – режим отпуска, приветственные сообщения

## 🛠️ Технологии

- **Python 3.11**
- **Django 4.2** (админка, ORM, миграции)
- **VK API** (vk-api библиотека)
- **SQLite** (база данных, хранится в Docker volume)
- **Docker + Docker Compose** (запуск в двух контейнерах: web и bot)
- **Gunicorn** (WSGI-сервер)

## 📦 Требования

- Сервер с **Docker** и **Docker Compose** (Raspberry Pi, VPS, любой Linux)

## 🚀 Быстрый старт (Docker)

### 1. Клонируйте репозиторий

```bash
git clone https://github.com/your-username/laser-crm.git
cd laser-crm
```

### 2. Создайте файл `.env` и заполните его

```env
VK_TOKEN=ваш_токен_сообщества
VK_GROUP_ID=123456789
SECRET_KEY=сгенерируйте_случайную_строку
DEBUG=False
```

### 3. Запустите контейнеры

```bash
docker compose up -d --build
```

### 4. Примените миграции и создайте суперпользователя

```bash
docker compose exec web python manage.py migrate
docker compose exec web python manage.py createsuperuser
```

### 5. Откройте админку

`http://ваш_сервер:8000/admin`

Войдите с логином и паролем, созданными на предыдущем шаге.

### 6. Настройте бота

- Добавьте **услуги** (раздел `Price list`) – укажите название, тип расчёта, базовую цену, единицу измерения и минимальную стоимость заказа.
- При необходимости добавьте **промокоды** и **материалы** на склад.

Теперь ваш VK-бот готов отвечать клиентам!

## 🔧 Ручная установка (без Docker)

Если вы хотите запустить проект без Docker (например, для разработки):

```bash
# Установите зависимости
pip install -r requirements.txt

# Примените миграции
python manage.py migrate

# Создайте суперпользователя
python manage.py createsuperuser

# Запустите сервер (админка)
python manage.py runserver

# В отдельном терминале запустите бота
python manage.py run_bot
```

## 💬 Чат с клиентом

Админка включает **встроенный чат**, стилизованный под мессенджер. Чтобы ответить клиенту:

1. Перейдите в раздел `Chat messages`.
2. Выберите сообщение и нажмите **«Ответить»**.
3. Введите текст и отправьте – сообщение уйдёт в VK и сохранится в истории.

Чат автоматически обновляется каждые 5 секунд.

## 🧪 11 типов калькулятора

Поддерживаются следующие алгоритмы расчёта стоимости:

| Тип | Описание | Параметры |
|-----|----------|-----------|
| `fixed` | Фиксированная цена за штуку | количество |
| `area_cm2` | Гравировка по площади (см²) | длина, ширина |
| `photo_raster` | Растровая гравировка фото | длина, ширина |
| `meter_thickness` | Резка по контуру (метры, толщина) | метры, толщина (мм) |
| `per_minute` | Повременная гравировка | минуты |
| `per_char` | За символ (текст на изделиях) | количество символов |
| `vector_length` | Длина вектора (для промышленной резки) | метры |
| `setup_batch` | Настройка + тираж (B2B) | стоимость настройки, цена за шт, количество |
| `cylindrical` | Цилиндрическая гравировка (ротационная ось) | диаметр (см), длина (см) |
| `volume_3d` | 3D-гравировка (глубокая выборка) | площадь (см²), глубина (мм) |
| `complex` | Материал + резка | стоимость материала, метры реза, ставка за метр |

Вы можете добавлять новые услуги через админку – бот автоматически подхватит их.
